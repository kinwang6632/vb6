中興商業銀行集中轉帳磁片檔案欄位說明
  1. 磁片規格 : MS-DOS format , ASCII code , 3.5 吋
  2. 檔案長度 : 80 bytes
  3. 磁片封面 : 寫明 公司名稱、公司統一編號、總金額、總筆數
  4. 檔案名稱 : 入帳 (貸方) CSBCR.DAT         扣帳 (借方) CSBDR.DAT
                       券商查詢(CSB.INQ)                 證券商查詢結果(CSB.REP)
  5. 檔案儲存 : 一磁片只存一入帳或扣帳檔
  6. 注意事項 : 資料型態為英、數字，請勿鍵入中文資料。欄位無資料時，請鍵入空白。
  7. 附 註： 客戶可將測試磁片直接交給主辦行做轉帳磁片基本檢核，並印出全部檢核資料， 並告知客戶磁片資料及欄位是否正確。
